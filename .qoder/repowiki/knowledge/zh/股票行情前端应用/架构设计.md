- **技术栈**：采用 Vue 3 (Composition API) + TypeScript + Vite 构建，使用 Pinia 进行状态管理，Vue Router 处理路由，Element Plus 作为 UI 组件库，ECharts 实现金融图表渲染。
- **分层架构**：遵循标准的 MVVM 模式。`src/api` 封装 Axios 实例与后端接口交互；`src/stores` (Pinia) 管理全局状态（如行情列表、自选股）；`src/pages` 承载业务视图逻辑；`src/components` 提供可复用 UI 组件。
- **路由与布局**：`src/router/index.ts` 定义客户端路由，支持懒加载页面组件。`App.vue` 作为根组件挂载 `router-view`，全局样式在 `App.vue` 中通过 CSS 变量定义深色主题。
- **部署架构**：采用多阶段 Docker 构建，生产环境使用 Nginx 托管静态资源，并配置反向代理将 `/api` 请求转发至后端服务，同时支持 WebSocket 连接升级。
