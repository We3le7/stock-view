- 所有页面组件均采用 `<script setup lang="ts">` 语法糖编写，结合 Composition API 组织逻辑。
- API 请求统一在 `src/api/index.ts` 中通过 Axios 实例封装，并按业务模块（quote, stock, watchlist, ai）导出对象。
- 状态管理使用 Pinia 的 `defineStore` 函数式写法，将 state、actions 封装在 setup 函数中返回。
- 金融数据涨跌颜色统一通过 CSS 变量 `--color-up` (红)、`--color-down` (绿) 及辅助函数 `getColor` 动态绑定样式。
- 页面级组件通过 `onMounted` 和 `onUnmounted` 生命周期钩子管理定时刷新任务（如 10 秒轮询行情）及资源释放。
