- **容器化编排**：利用 `docker-compose.yml` 定义 PostgreSQL、Redis、Backend (FastAPI) 和 Frontend (Vue 3) 的服务依赖与网络拓扑，实现一键式环境部署。
- **共享基础设施契约**：后端与前端通过统一的环境变量模板 (`.env.example`) 协调数据库连接、Redis 缓存策略及 AI 适配器配置，确保开发与生产环境的一致性。
- **前后端分离架构**：前端通过 Nginx 静态托管并代理 API 请求至后端服务，后端负责核心业务逻辑、异步数据采集及 WebSocket 实时推送，两者通过 RESTful API 与 WebSocket 协议进行松耦合通信。
