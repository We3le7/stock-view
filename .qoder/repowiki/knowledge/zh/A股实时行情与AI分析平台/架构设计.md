- **分层架构**：后端采用 FastAPI 构建 RESTful API 与 WebSocket 服务，前端基于 Vue 3 + TypeScript + Pinia 实现响应式交互。
- **数据流设计**：通过 Celery Beat 定时触发数据采集器（东方财富/新浪财经），数据经 Redis 缓存与 Pub/Sub 广播至 WebSocket 服务，最终推送到前端；历史数据持久化至 PostgreSQL。
- **模块化设计**：核心模块包括 `api`（路由层）、`services/collector`（多源采集适配层）、`ai`（策略抽象与适配器模式）以及 `core`（配置与基础设施连接）。
- **接口边界**：定义清晰的 `BaseCollector` 与 `AIInterface` 抽象基类，支持数据源故障转移与 AI 分析策略的动态切换（Mock/规则引擎/ML）。
